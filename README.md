# Web San Xuat Video Giao Duc AI

Day la mot web app tinh chay bang `HTML + CSS + JavaScript` thuan, khong can cai dat thu vien de mo va dung nhanh.

## Tinh nang

- Quy trinh 6 buoc: cau hinh du an, tao kich ban, chon nhan vat, phan tach canh, tao prompts, xuat ban.
- Tu luu `localStorage` ngay trong trinh duyet.
- Tao kich ban mau tu brief.
- Tach canh tu dong theo thoi luong video.
- Tao prompt pack cho tung canh.
- Xuat `JSON` va `Markdown`, tu dong an API key khi export.

## Cach chay

1. Mo file `index.html` bang trinh duyet.
2. Hoac chay server tinh bat ky neu ban muon preview on dinh hon.

Vi du voi PowerShell:

```powershell
python -m http.server 5500
```

Sau do truy cap `http://localhost:5500`.
